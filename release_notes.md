### Hustle Link Demo - Version v1.0.2

This is a pre-release demo showcasing the core functionality of the Hustle Link app.

**New Features in this Demo:**

* Firebase-powered user registration and login.
* Role-based access for Hustlers and Employers.
* Hustlers can view a personalized job feed based on their skills.
* Employers can post new job opportunities.
* Enhanced UI/UX improvements and bug fixes.

**How to install:**
Download the `app-release.apk` file and install it on your Android device.

Please report any bugs or issues on the GitHub Issues page.
